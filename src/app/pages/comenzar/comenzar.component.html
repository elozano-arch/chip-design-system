Comenzar

Guía paso a paso para integrar el Design System CHIP 2.0 al proyecto Angular. Seguir estas instrucciones para configurar el entorno, instalar dependencias y empezar a construir interfaces consistentes con la identidad visual de la CGN.

1 Prerrequisitos

Antes de iniciar, asegúrate de tener instaladas las siguientes herramientas y versiones en tu máquina de desarrollo.

Node.js 22 Entorno de ejecución de JavaScript
Angular 19 Framework de desarrollo frontend
PrimeNG 19 Biblioteca de componentes UI para Angular

2 Instalación

Crea un nuevo proyecto Angular y agrega las dependencias necesarias. Si ya tienes un proyecto existente, puedes saltar la creación e ir directamente a la instalación de PrimeNG.

a) Crea un nuevo proyecto con Angular CLI usando SCSS como preprocesador de estilos.
b) Instala PrimeNG 19, el paquete de temas @primeng/themes y los íconos primeicons.
c) Copia el archivo chip-preset.ts (disponible en la página de Inicio) dentro de la carpeta src/app/theme/ de tu proyecto.

3 Configuración del Preset

Registra el ChipPreset en el archivo app.config.ts usando el provider providePrimeNG. Esto aplicará la identidad visual del Kit UI v9.2 (GOV.CO) a todos los componentes de PrimeNG automáticamente.

4 Estilos Globales

Configura el archivo src/styles.scss con las fuentes, variables CSS y los estilos base del Design System. Estas variables estarán disponibles en todos los componentes del proyecto.

5 Estructura de Carpetas Recomendada

Organiza el proyecto siguiendo esta estructura para mantener la consistencia y facilitar el mantenimiento a largo plazo.

6 Importar y Usar Componentes

Cada componente de PrimeNG se importa individualmente como un módulo standalone. A continuación se muestra un ejemplo completo de cómo crear una página que utiliza botones, tablas y etiquetas.

Importa solo los módulos que necesitas en cada componente para optimizar el bundle.
Usa standalone: true en todos los componentes para aprovechar la arquitectura moderna de Angular 19.
Consulta la sección de Ejemplos de Uso para ver patrones completos de formularios, tablas y diálogos.
